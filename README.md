# Sales Analytics Dashboard

Built an interactive Sales Analytics Dashboard analyzing 200+ sales 
transactions across 12 months (Jan–Dec 2024).

## Tools Used
Power BI | Microsoft Excel

## What's Inside
- KPI Cards: Total Sales, Total Orders, Units Sold, Avg Order Value
- Slicers: Month, Region, Category, Salesperson
- Charts: Monthly Sales Trend, Category Donut, Sales by Region, Top Products
- Salesperson Performance Table

## Key Insights
- Laptop was the highest revenue-generating product
- Electronics contributed the most to overall sales
- Dynamic slicers allow drill-down by region, category and salesperson

## Files in This Repository
- sales-dashboard.pbix — Power BI dashboard file
- sales-data.xlsx — Raw dataset used for analysis
